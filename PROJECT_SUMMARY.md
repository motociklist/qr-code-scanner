# QR Code Scanner - Полноценное приложение с подписками и рекламой

## Обзор проекта

Полноценное приложение для сканирования и создания QR-кодов с интеграцией подписок, рекламы и аналитики. Проект готов к публикации в App Store и Google Play.

## Реализованные функции

### Основной функционал

1. **Сканирование QR-кодов**
   - Мгновенное сканирование с камеры
   - Сканирование из галереи
   - Автоматическое распознавание различных типов QR-кодов

2. **Создание QR-кодов**
   - URL
   - Текст
   - Телефон
   - Email
   - Контакт (vCard)
   - WiFi

3. **История сканирований**
   - Сохранение всех отсканированных QR-кодов
   - Фильтрация по типу
   - Группировка по датам

4. **Мои QR-коды**
   - Сохранение созданных QR-кодов
   - Поиск по названию
   - Экспорт и удаление

5. **Обработка результатов**
   - Открытие URL
   - Звонок по телефону
   - Отправка email
   - Добавление контактов
   - Подключение к WiFi (информация)
   - Копирование и отправка

### Монетизация

1. **Подписки (In-App Purchase)**
   - Недельная подписка
   - Месячная подписка
   - Годовая подписка
   - Восстановление покупок
   - Проверка статуса подписки

2. **Реклама (Google Mobile Ads)**
   - Баннерная реклама
   - Межстраничная реклама
   - Реклама с вознаграждением
   - Отключение рекламы для премиум пользователей

3. **Аналитика (Firebase Analytics)**
   - Отслеживание сканирований
   - Отслеживание создания QR-кодов
   - Отслеживание покупок подписок
   - Отслеживание показа рекламы

## Технологии

- **Flutter** - кроссплатформенный фреймворк
- **mobile_scanner** - сканирование QR-кодов
- **qr_flutter** - генерация QR-кодов
- **in_app_purchase** - подписки
- **google_mobile_ads** - реклама
- **firebase_analytics** - аналитика
- **shared_preferences** - локальное хранилище
- **contacts_service** - работа с контактами

## Настройка перед публикацией

### 1. Firebase
- Создайте проект в Firebase Console
- Добавьте конфигурационные файлы для iOS и Android
- Включите Firebase Analytics

### 2. Google Mobile Ads
- Создайте приложение в AdMob
- Создайте рекламные блоки
- Замените тестовые Ad Unit IDs на реальные

### 3. In-App Purchase
- Настройте продукты в App Store Connect (iOS)
- Настройте продукты в Google Play Console (Android)
- Убедитесь, что идентификаторы продуктов совпадают в коде

### 4. Разрешения
- Настройте разрешения для камеры, галереи и контактов
- Добавьте описания разрешений для iOS
